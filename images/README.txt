This folder will contain your brand logo and other images.

📁 Recommended Images:

1. logo.png
   - Your Tropical Charm logo
   - Size: 500x300px (or similar ratio)
   - Format: PNG with transparent background
   - Used in: Navigation bar, homepage hero

2. favicon.png
   - Website icon (appears in browser tab)
   - Size: 32x32px or 64x64px
   - Format: PNG

3. products/ folder
   - Create a subfolder called "products"
   - Add product images here
   - Recommended size: 800x800px
   - Format: JPG or PNG
   
   Example filenames:
   - crystal-bloom.jpg
   - ocean-blue.jpg
   - tropical-forest.jpg
   - sandstone.jpg
   - rose-blossom.jpg
   - sun-charm.jpg

4. about/ folder (optional)
   - Brand story images
   - Behind-the-scenes photos
   - Team photos

Tips:
- Keep file sizes optimized (under 500KB per image)
- Use descriptive filenames (no spaces, use hyphens)
- Maintain consistent image dimensions
- Use high-quality images for best impression
