## 📞 Support Reminders

### When You Need Help
1. Check the documentation files first
2. Review code comments
3. Search the specific error message
4. Check browser console for errors (F12)

### Common Issues Solutions
- Server won't start → Check Node.js installed
- Port in use → Change port number
- Images not showing → Check file paths
- Surprise expired → Update date in surprises.json
